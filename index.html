Coqrel: binary logical relations for Coq

Introduction

Coqrel is a library for the Coq proof assistant which facilitates relational reasoning by mechanizing techniques associated with binary logical relations. It provides a unified language and methodology for building relations between complex objects, and an extensible tactic library for solving relational goals.

Documentation

I presented Coqrel at the CoqPL'15 workshop; an extended abstract is available.

While additionnal documentation is scarce, the code is well-commented and should be reasonably easy to read. Coqrel uses the Coq's typeclass system extensively, both to make the system extensible by the user, and to enable full backtracking between the different components. As a result, some familiarity with the typeclass system of Coq will be valuable to understand the code.

You can browse the Coqdoc table of contents and index here. The top-level file LogicalRelations is a good place to start.
